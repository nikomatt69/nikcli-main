# Coinbase AgentKit Tool Documentation

## Purpose
Official Coinbase AgentKit integration for blockchain operations.

## Best Practices
- Initialize before use; validate env keys
- Confirm transfers via chat flow

## Enterprise Guidance (Network/Exec-like Operations)
- Preflight annotates operationType=network; medium/high risk for transfers.
- Approval required for transactions; session approval optional.
- Always display asset, amount, recipient, and network in approval.
