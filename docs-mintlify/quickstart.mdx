# Quickstart Guide

## 1. Install

```bash
# Universal installer
curl -fsSL https://raw.githubusercontent.com/nikomatt69/nikcli-main/main/installer/install.sh | bash

# Manual global install
bun add -g @nicomatt69/nikcli
nikcli
```

## 2. Setup API Key

```
/set-key claude-sonnet \"sk-ant-...\"
# Or env: export ANTHROPIC_API_KEY=sk-ant-...
```

## 3. First Session

```bash
nikcli
/help          # Commands list
/agents        # List agents
/model claude-sonnet
/agent universal-agent \"Hello NikCLI!\"
```

## Interactive Demo

```
Chat with NikCLI:
/plan \"Create a React app\"

Follow the autonomous plan!
```
