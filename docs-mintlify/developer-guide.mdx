## Setup

```bash
git clone https://github.com/nikomatt69/nikcli-main
cd nikcli-main
bun install
bun build
bun start
```

## Scripts (from package.json)

- `bun dev` - Watch mode
- `bun test` - Vitest
- `bun lint` - Biome
- `bun build:binary` - pkg binaries
