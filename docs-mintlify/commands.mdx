# /commands Reference

All commands are case-insensitive. Type `/help` for overview.

## Core Commands

| Command | Description | Example |
|---------|-------------|---------|
| `/help` | Show help | `/help` |
| `/quit` | Exit | `/exit` |
| `/model <name>` | Switch model | `/model claude-sonnet` |
| `/set-key <provider> <key>` | Set API key | `/set-key openai sk-...` |
| `/config` | Show config | `/config` |
| `/agents` | List agents | `/agents` |
| `/agent <name> <task>` | Run agent | `/agent universal-agent \"code review\"` |

## File & Project

| Command | Description |
|---------|-------------|
| `/read <file>` | Read file |
| `/write <file> <content>` | Write file |
| `/edit <file>` | Interactive edit |
| `/ls` | List files |
| `/grep <pattern>` | Search code |
| `/build` | Run build script |

## VM Management (20+)

| `/vm-create <repo>` | Create VM container |
| `/vm-list` | List VMs |
| `/vm-connect <id>` | Connect to VM |

## Web3 & Tools

- `/web3 init`, `/goat chat \"balance\"`
- `/browser https://example.com`
- `/analyze-image image.png`
