## Overview

Quick search input field.

## Import

```ts
import { SearchBox } from 'nikcli-tui';
```

## Usage

```ts
new SearchBox({ parent: screen, label: 'Search' });
```

- onChange?: (value: string) => void â€” Typing callback.
- onSearch?: (value: string) => void â€” Enter/submit callback.
- value?: string â€” Controlled value.
- clearable?: boolean â€” Shows a clear button (default true).
- searchIcon?: string â€” Optional leading icon (default 'ðŸ”').
- width?: string | number â€” Container width (default 100%).
- height?: number â€” Container height (default 3).

## Methods

- getValue(), setValue(value), clear().
- setPlaceholder(text), focus(), blur().
- setSearchIcon(icon), setClearable(bool).
- search(): trigger onSearch with current value.
- getDimensions(), setDimensions(w, h).

## Keyboard

- Enter: triggers onSearch if provided.
