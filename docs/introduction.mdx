## What is NikCLI?

NikCLI is a **next-generation, AI-powered development assistant** that fuses the power of artificial intelligence with a comprehensive suite of practical development tools. Unlike traditional CLI tools, NikCLI features **cognitive orchestration**‚Äîan advanced system that understands natural language tasks, breaks them down intelligently, and executes them using the most appropriate strategies.

### Core Philosophy

NikCLI's design is guided by a core philosophy of intelligent automation and continuous improvement.

<Steps>
  <Step title="Cognitive Understanding">
    NikCLI leverages advanced NLP to understand your intent, extract key entities, and accurately assess task complexity.
  </Step>
  <Step title="Intelligent Planning">
    It creates dynamic orchestration plans with optimal strategies, including sequential, parallel, hybrid, and adaptive execution.
  </Step>
  <Step title="Adaptive Execution">
    Tasks are executed with real-time supervision and automatic fallback strategies to ensure resilience and success.
  </Step>
  <Step title="Continuous Learning">
    NikCLI learns from every interaction to enhance future task execution, planning, and decision-making.
  </Step>
</Steps>

### ü§ñ Dual Agent Architecture

NikCLI's power lies in its unique dual-agent architecture, which combines cognitive prowess with secure execution.

<AccordionGroup>
  <Accordion title="Universal Agent - The Cognitive Powerhouse">

## Getting Started

Ready to revolutionize your development workflow? Get started with NikCLI in just a few simple steps.

<Steps>
  <Step title="Install NikCLI">
    Our universal installer automatically detects your package manager and installs NikCLI.
    ```bash
    # Universal installer
    curl -fsSL https://raw.githubusercontent.com/nikomatt69/nikcli-main/main/installer/install.sh | bash
    ```
  </Step>
  <Step title="Configure API Keys">
    Securely configure your AI provider API keys. NikCLI encrypts and stores them with AES-256-GCM.
    ```bash
    nikcli
    /set-key anthropic sk-ant-api03-...
    /set-key openai sk-...
    ```
  </Step>
  <Step title="Start Your First Task">
    Begin with a simple task and let NikCLI's Universal Agent handle the rest.
    ```bash
    /agent universal-agent "Create a simple React component with TypeScript and add a unit test."
    ```
  </Step>
</Steps>
