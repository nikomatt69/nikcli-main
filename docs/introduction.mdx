# NikCLI

NikCLI is a powerful AI-powered CLI tool that combines natural language understanding with comprehensive development capabilities. Built with TypeScript and the Vercel AI SDK, it provides intelligent code analysis, generation, and autonomous development workflows.

## Key Features

### Multi-Provider AI Support

NikCLI supports multiple AI providers with automatic model routing:

| Provider | Models | Context Window |
|----------|--------|----------------|
| **Anthropic** | Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku, Claude Sonnet 4 | 200K tokens |
| **OpenAI** | GPT-5, GPT-4o, GPT-4o Mini, GPT-3.5 Turbo | 128K-200K tokens |
| **Google** | Gemini 2.5 Pro/Flash, Gemini 2.0, Gemini 1.5 Pro | Up to 2M tokens |
| **Groq** | High-speed inference models | Varies |
| **OpenRouter** | 100+ models with reasoning support | Varies |
| **Ollama** | Local models (Llama, Mistral, CodeLlama) | Varies |

### Agent System

NikCLI includes a sophisticated multi-agent architecture:

- **Universal Agent**: All-in-one enterprise agent with 70+ capabilities
- **VM Agent**: Secure virtualized execution with API key isolation
- **Specialized Agents**: React, Backend, DevOps, Code Review, and more
- **Dynamic Agents**: Create custom agents from blueprints

### Operating Modes

| Mode | Description | Command |
|------|-------------|---------|
| **Default** | Conversational chat with auto-todos | `/default` |
| **Plan** | Structured planning with approval workflows | `/plan [task]` |
| **VM** | Virtual machine development environment | `/vm` |

## Quick Example

```bash
# Install NikCLI
npm install -g @nicomatt69/nikcli

# Set your API key
nikcli /set-key anthropic sk-ant-...

# Start coding
nikcli

# Use an agent
/agent react-expert "Create a login form with validation"

# Run autonomous mode
/auto "Build a REST API with authentication"
```

## System Requirements

- **Node.js**: v18.0.0 or higher
- **npm/yarn/pnpm/bun**: Any modern package manager
- **API Key**: At least one AI provider key (Anthropic, OpenAI, Google, etc.)
