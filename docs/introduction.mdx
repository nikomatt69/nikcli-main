## What is NikCLI?

NikCLI is a **production-ready autonomous AI development assistant** that provides an intelligent command-line interface for software development. It combines conversational AI with autonomous code generation, file manipulation, and project management capabilities.

  </Tab>
  <Tab title="Ollama">
    Local models for privacy and offline usage
    ```bash
    # No API key required for local models
    # Ensure the Ollama daemon is running (default 127.0.0.1:11434)
    ollama serve

    # Inside NikCLI, switch model via slash command
    /model llama3.1:8b
    ```

## Quick Start

Get started with NikCLI in just a few minutes:

<Steps>
  <Step title="Install NikCLI">
    ```bash npm install -g @nicomatt69/nikcli ```
  </Step>
  <Step title="Set API Key">
    ```bash export ANTHROPIC_API_KEY="your-api-key" ```
  </Step>
  <Step title="Start Interactive Mode">```bash nikcli ```</Step>
  <Step title="Try Your First Command">``` > /help ```</Step>
</Steps>
