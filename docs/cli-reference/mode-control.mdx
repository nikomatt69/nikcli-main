### Autonomous Mode

Autonomous mode enables AI-driven execution with minimal supervision.

```bash
# Start autonomous execution
/auto "task description"

# Examples
/auto "create a todo application with React and Node.js"
/auto "implement user authentication with JWT tokens"
/auto "optimize application performance"
```

**Autonomous Features:**
- Automatic task execution
- Minimal user intervention
- Progress tracking
- Quality assurance

**Control Commands:**
```bash
# View progress and status
/stats      # Show usage statistics
/agents     # Show agent status

# Safety controls
/dev-mode   # Enable developer mode
/safe-mode  # Enable safe mode
/clear-approvals  # Clear cached approvals
```

### VM Mode

VM mode provides secure, isolated development environments using Docker containers.

```bash
# Switch to VM mode
/vm
/vm-mode

# VM operations
/vm-create https://github.com/user/repo.git  # Create new container
/vm-list                                     # List active containers
/vm-connect [container-id]                   # Connect to container
/vm-create-pr [id] "title" "description"     # Create pull request
```

**VM Mode Features:**
- Docker container isolation
- Repository cloning and analysis
- Secure development environment
- Pull request creation capabilities
- API key security proxy

**VM Management Commands:**
```bash
/vm-stop [container-id]     # Stop container
/vm-remove [container-id]   # Remove container
/vm-logs [container-id]     # View container logs
```
