### üñ•Ô∏è Basic System Commands

| Command          | Description                                   | Usage              | Example    |
| ---------------- | --------------------------------------------- | ------------------ | ---------- |
| `/help`          | Show help message with all available commands | `/help`            | `/help`    |
| `/quit`, `/exit` | Exit the chat interface                       | `/quit` or `/exit` | `/quit`    |

### üß† Model Management

<AccordionGroup>
  <Accordion title="/model - Switch AI Models">
    Switch between different AI models or show current model. ```bash # Show
    current model /model # Switch to specific model /model claude-3-5-sonnet
    /model gpt-4 /model gemini-pro /model ollama:llama2 ``` **Examples:** -
    Anthropic: `claude-sonnet-4-20250514`, `claude-3-5-sonnet-latest` - OpenAI:
    `gpt-5`, `gpt-4o-mini` - Google: `gemini-1.5-pro` - Ollama: `llama3.1:8b`,
    `mistral:7b`
  </Accordion>

{" "}
<Accordion title="/models - List Available Models">
  Display all available models with their status and capabilities. ```bash
  /models ``` **Output includes:** - Model name and provider - Availability
  status - Token limits - Current API key status
</Accordion>

  <Accordion title="/set-key - Configure API Keys">
    Set API keys for different AI providers. ```bash /set-key anthropic
    sk-ant-api03-... /set-key openai sk-... /set-key google AIza... ```
    <Warning>
      API keys are read from your `~/.nikcli/config.json` or environment vars.
    </Warning>
  </Accordion>
</AccordionGroup>

## Operating Modes

NikCLI supports three primary operating modes accessible via `Shift+Tab`:

<Tabs>
  <Tab title="üí¨ Default Chat">
    **Standard Mode** - Interactive conversation - Manual confirmation for
    operations - Step-by-step guidance - Best for learning and experimentation
  </Tab>

{" "}
<Tab title="üìã Planning Mode">
  **Strategic Planning** - Comprehensive execution plans - Todo.md file
  generation - Approval workflows - Best for complex projects
</Tab>

  <Tab title="ü§ñ Auto Mode">
    **Autonomous Execution** - Automatic task execution - Minimal user
    intervention - Streamlined workflows - Best for rapid development
  </Tab>
</Tabs>

## Keyboard Shortcuts

| Shortcut    | Description                                  |
| ----------- | -------------------------------------------- |
| `Ctrl+C`    | Stop current operation or streaming response |
| `/`         | Open command menu (when not processing)      |
| `Shift+Tab` | Cycle between operating modes                |
| `ESC`       | Return to default chat mode                  |
| `Tab`       | Auto-complete commands and file paths        |
