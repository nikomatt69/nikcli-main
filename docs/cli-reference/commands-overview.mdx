# CLI Commands Overview

NikCLI provides 150+ interactive slash commands organized into functional categories. All commands are prefixed with `/`.

## Command Categories

### System Commands

| Command | Description | Example |
|---------|-------------|---------|
| `/help` | Show comprehensive help | `/help` |
| `/exit`, `/quit` | Exit NikCLI | `/exit` |
| `/clear` | Clear session context | `/clear` |
| `/status` | Show system status | `/status` |
| `/debug` | Show debug information | `/debug` |
| `/init [--force]` | Initialize project context | `/init` |
| `/dashboard` | Toggle analytics dashboard | `/dashboard start` |

## Prerequisites

Some commands require additional setup:

- **Web3**: Wallet configuration and API keys
- **Figma**: Figma access token
- **Background agents**: nikd daemon running
- **VM commands**: Docker installation
- **Image commands**: Vision model access
