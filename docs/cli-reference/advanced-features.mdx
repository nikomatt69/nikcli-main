## Workflow Automation

### `/workflow` - Custom Workflow Management

<Tabs>
  <Tab title="Workflow Creation">
    ```bash # Create new workflow /workflow create "full-stack-setup" /workflow
    create --template api-development --customize # Define workflow steps
    /workflow define --steps " 1. analyze-project 2. setup-backend:backend-agent
    3. create-frontend:react-expert 4. integrate-apis:universal-agent 5.
    run-tests:all " ```
  </Tab>

  <Tab title="Workflow Execution">
    ```bash # Execute workflow /workflow run full-stack-setup /workflow run
    --with-approval api-development # Conditional workflow execution /workflow
    run deployment --if "tests-passing" /workflow run optimization --unless
    "performance-acceptable" ```
  </Tab>

  <Tab title="Workflow Management">
    ```bash # List workflows /workflow list --detailed --usage-stats # Share
    workflows /workflow export full-stack-setup --format yaml /workflow import
    team-workflow.yml --validate # Workflow versioning /workflow version --tag
    v1.2.0 --changelog "Added testing step" ```
  </Tab>
</Tabs>

### Advanced Scripting

<AccordionGroup>
  <Accordion title="Batch Operations">
    ```bash # Batch file operations /batch process "src/**/*.ts" --operation
    "add-type-annotations" /batch refactor "components/*.tsx" --pattern
    "class-to-hooks" # Batch agent execution /batch agents --task
    "optimize-performance" --files "*.tsx" /batch parallel --agents
    "react-expert,backend-agent" --tasks tasks.json ```
  </Accordion>

  <Accordion title="Pipeline Operations">
    ```bash # Create processing pipeline /pipeline create code-quality --steps "
    lint → format → test → security-scan → deploy " # Execute pipeline with data
    flow /pipeline run code-quality --input "src/" --output "dist/" /pipeline
    run --parallel optimization --chunks 4 ```
  </Accordion>

  <Accordion title="Conditional Logic">
    ```bash # Conditional command execution /if file-exists "package.json" then
    run "npm install" /if git-has-changes then agent react-expert "review
    changes" # Loop operations /for-each component in "src/components/*.tsx" do
    optimize /while tests-failing do fix-issues --auto-retry 3 ```
  </Accordion>
</AccordionGroup>

### Custom Commands

<AccordionGroup>
  <Accordion title="Command Registration">
    ```typescript
    // Register custom command
    import { registerCommand } from '@nikcli/plugin-api';
    
    registerCommand('deploy-staging', {
      description: 'Deploy to staging environment',
      parameters: {
        branch: { type: 'string', default: 'main' },
        skipTests: { type: 'boolean', default: false }
      },
      handler: async (params, context) => {
        // Custom deployment logic
        await context.agents.devops.execute('deploy', params);
      }
    });
    ```
  </Accordion>
  
  <Accordion title="Agent Extensions">
    ```typescript
    // Extend existing agent
    import { extendAgent } from '@nikcli/plugin-api';
    
    extendAgent('react-expert', {
      capabilities: ['styled-components', 'emotion', 'chakra-ui'],
      tools: ['component-generator', 'style-optimizer'],
      prompts: {
        'create-styled-component': './prompts/styled-component.txt'
      }
    });
    ```
  </Accordion>
</AccordionGroup>

### SDK Development

<Tabs>
  <Tab title="JavaScript SDK">
    ```typescript
    import { NikCLI } from '@nikcli/sdk';
    
    const cli = new NikCLI({
      apiKey: process.env.NIKCLI_API_KEY,
      baseUrl: 'http://localhost:3001'
    });
    
    // Execute agent task
    const result = await cli.agents.execute('react-expert', {
      task: 'create login component',
      context: './src/components/'
    });
    ```
  </Tab>
  
  <Tab title="Python SDK">
    ```python
    from nikcli import NikCLI
    
    cli = NikCLI(
        api_key=os.environ['NIKCLI_API_KEY'],
        base_url='http://localhost:3001'
    )
    
    # Execute workflow
    result = cli.workflows.execute('deployment', {
        'environment': 'staging',
        'branch': 'main'
    })
    ```
  </Tab>
  
  <Tab title="Go SDK">
    ```go
    package main
    
    import "github.com/nikcli/go-sdk"
    
    client := nikcli.NewClient(nikcli.Config{
        APIKey: os.Getenv("NIKCLI_API_KEY"),
        BaseURL: "http://localhost:3001",
    })
    
    // Create session
    session, err := client.Sessions.Create("API Development")
    if err != nil {
        log.Fatal(err)
    }
    ```
  </Tab>
</Tabs>
