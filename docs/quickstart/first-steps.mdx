# First Steps

This guide covers the essential commands to get started with NikCLI.

## Starting NikCLI

```bash
# Start in current directory
nikcli

# Start with a specific model
nikcli --model claude-3-5-sonnet
```

## Essential Commands

### Getting Help

```bash
/help              # Show comprehensive help
/commands          # Alias for /help
```

### Model Management

```bash
/models            # List all available models
/model <name>      # Switch to specific model
/temp 0.7          # Set temperature (0.0-2.0)
```

### File Operations

```bash
/read <file>       # Read file contents
/write <file> <content>  # Write to file
/edit <file>       # Edit file interactively
/ls [directory]    # List files
/search <query>    # Search in files
/grep <query>      # Alias for search
```

### Terminal Commands

```bash
/run <command>     # Execute any command
/npm <args>        # Run npm commands
/yarn <args>       # Run yarn commands
/git <args>        # Run git commands
/docker <args>     # Run docker commands
/build             # Build project
/test [pattern]    # Run tests
```

### Agent Commands

```bash
/agents            # List all agents
/agent <name> <task>  # Run specific agent
/auto <task>       # Autonomous execution
/parallel <agents> <task>  # Run agents in parallel
```

## Working with Agents

### Available Agents

| Agent | Description |
|-------|-------------|
| `universal-agent` | All-purpose development (70+ capabilities) |
| `react-expert` | React and frontend development |
| `backend-expert` | API and backend development |
| `devops-expert` | DevOps and infrastructure |
| `code-review` | Code quality analysis |
| `autonomous-coder` | Fully autonomous coding |
| `frontend-expert` | Frontend/UI development |
| `nextjs-expert` | Next.js specialist |
| `vue-expert` | Vue.js specialist |
| `framer-motion-expert` | UI animations |
| `system-admin` | System administration |
| `optimization-expert` | Performance optimization |
| `ai-sdk-integrator` | Vercel AI SDK integration |

### Using an Agent

```bash
# React component
/agent react-expert "Create a responsive navbar with dark mode"

# API endpoint
/agent backend-expert "Create REST API for authentication"

# Code review
/agent code-review "Review src/auth/login.ts for security"

# Autonomous task
/auto "Build a todo app with React and Express"
```

## Operating Modes

### Default Mode
Standard conversational chat with auto-todos for complex tasks.
```bash
/default
```

### Plan Mode
Structured planning with approval before execution.
```bash
/plan "Implement user authentication with JWT"
```

### VM Mode
Isolated development in Docker containers.
```bash
/vm
/vm-create ubuntu
/vm-connect <id>
```

## Session Management

```bash
/new [title]       # Start new session
/sessions          # List sessions
/export [id]       # Export to markdown
/resume [id]       # Resume work session
/save-session [name]  # Save current session
```

## Practical Examples

### Create a Component

```bash
/agent react-expert "Create DataTable with sorting and pagination"
```

### Analyze Code

```bash
/search "async function" src/
/read src/services/auth.ts
/agent code-review "Review src/services/auth.ts"
```

### Git Workflow

```bash
/git status
/git checkout -b feature/auth
/git commit -m "Add authentication"
```

## Memory & Context

```bash
/remember "fact"   # Store in long-term memory
/recall "query"    # Search memories
/memory stats      # Show memory statistics
/context           # Manage context
/index <path>      # Index files for context
```

## Undo/Redo

```bash
/undo [count]      # Undo last N edits
/redo [count]      # Redo last N edits
/edit-history      # Show edit history
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Cancel operation |
| `Ctrl+D` | Exit NikCLI |
| `Up/Down` | Navigate history |
| `Tab` | Autocomplete |
