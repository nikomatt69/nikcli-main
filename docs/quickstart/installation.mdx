# Installation

NikCLI can be installed via npm, yarn, pnpm, or bun.

## Requirements

- **Node.js**: v18.0.0 or higher
- **Package Manager**: npm, yarn, pnpm, or bun
- **Operating System**: macOS, Linux, or Windows

## Quick Install

<Tabs>
  <Tab title="npm">
    ```bash
    npm install -g @nicomatt69/nikcli
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn global add @nicomatt69/nikcli
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add -g @nicomatt69/nikcli
    ```
  </Tab>
  <Tab title="bun">
    ```bash
    bun add -g @nicomatt69/nikcli
    ```
  </Tab>
</Tabs>

## Universal Installer Script

<Tabs>
  <Tab title="Unix/macOS">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/nikomatt69/nikcli-main/main/installer/install.sh | bash
    ```
  </Tab>
  <Tab title="Windows PowerShell">
    ```powershell
    iwr -useb https://raw.githubusercontent.com/nikomatt69/nikcli-main/main/installer/install.ps1 | iex
    ```
  </Tab>
</Tabs>

## Install from Source

```bash
# Clone repository
git clone https://github.com/nikomatt69/nikcli-main
cd nikcli-main

# Install dependencies
pnpm install

# Build
pnpm run build

# Link globally
npm link
```

## Verify Installation

```bash
# Check version
nikcli --version

# Expected output
NikCLI v1.6.0
```

## API Key Setup

NikCLI requires at least one AI provider API key:

```bash
# Start NikCLI
nikcli

# Set your preferred provider key
/set-key anthropic sk-ant-...
/set-key openai sk-...
/set-key google AIza...
/set-key openrouter sk-or-v1-...
/set-key groq gsk_...
/set-key cerebras <key>
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Permission Errors">
    ```bash
    # Option 1: Fix npm permissions (recommended)
    mkdir ~/.npm-global
    npm config set prefix '~/.npm-global'
    echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
    source ~/.bashrc
    npm install -g @nicomatt69/nikcli

    # Option 2: Use npx
    npx @nicomatt69/nikcli
    ```
  </Accordion>

  <Accordion title="Node.js Version Issues">
    ```bash
    # Check version
    node --version

    # Update with nvm
    nvm install 18
    nvm use 18
    ```
  </Accordion>

  <Accordion title="Package Manager Conflicts">
    ```bash
    npm cache clean --force
    npm uninstall -g @nicomatt69/nikcli
    npm install -g @nicomatt69/nikcli
    ```
  </Accordion>
</AccordionGroup>

## Environment Variables

Optional environment variables for enhanced features:

```bash
# Redis/Upstash caching
export REDIS_URL="redis://..."
export UPSTASH_REDIS_REST_URL="https://..."
export UPSTASH_REDIS_REST_TOKEN="..."

# Supabase integration
export SUPABASE_URL="https://..."
export SUPABASE_ANON_KEY="..."

# Quiet startup
export NIKCLI_QUIET_STARTUP=true
```
